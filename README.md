# 介绍

电商后台管理软件，C/S架构，Delphi 7编写实现，Window系统平台运行。共分为如下模块：

DepotManagerCenter --------- 仓库管理

FinanceCenter ---------------------财务管理

LogisticsCenter---------------------物流管理

MemService-------------------------用户管理

ProductCenter----------------------商品管理

StockCenter--------------------------库存管理

SysManager--------------------------系统管理

支持RBAC权限管理模型，不同角色用户访问不同模块功能。

# 环境说明

支持 Window XP， Window 7， Window 10

# 部署
使用Delphi编译生成EXE可执行文件，直接在window系统上执行。
